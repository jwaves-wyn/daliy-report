# Genians 일일 업무보고서 자동 생성기

이 프로젝트는 Genians 기술지원팀의 일일 업무보고서 자동 생성을 연습하기 위한 기본 웹앱입니다.

## 포함 파일

- `messenger_chat_sample.txt`
- `jira_task_sample.csv`

위 두 파일은 참고용 샘플 데이터이며, 삭제하거나 이름을 바꾸지 않고 그대로 사용합니다.

## 새로 추가된 파일

- `index.html`
- `style.css`
- `app.js`
- `README.md`

## 현재 동작

- 작성일을 입력할 수 있습니다.
- 메신저 대화 내용을 붙여넣을 수 있습니다.
- JIRA CSV 데이터를 붙여넣을 수 있습니다.
- 샘플 데이터 불러오기 버튼으로 참고 파일 내용을 화면에 채울 수 있습니다.
- 로컬 샘플 보고서 생성 버튼으로 OpenAI API 없이 임시 업무보고서를 만들 수 있습니다.
- OpenAI API Key를 저장하면 AI 보고서 생성 버튼을 사용할 수 있습니다.
- 보고서 복사 버튼으로 생성된 보고서를 클립보드에 복사할 수 있습니다.
- 초기화 버튼으로 입력값과 결과를 지울 수 있습니다.

## 출력 형식

로컬 샘플 보고서는 다음 구조를 따릅니다.

- `[일일 업무 요약]`
- `[주요 완료 업무]`
- `[진행 중 업무]`
- `[지연/이슈 사항]`
- `[내일 예정 업무]`
- `[지원 필요 사항]`
- `[내부 공유 메모]`

## 실행 방법

브라우저에서 `index.html` 파일을 열면 됩니다.
샘플 데이터 불러오기는 브라우저와 실행 방식에 따라 로컬 웹 서버에서 더 안정적으로 동작합니다.

## 참고

이 단계에서는 OpenAI API 연결을 넣지 않았습니다.
화면 구조와 기본 동작만 구현되어 있습니다.
